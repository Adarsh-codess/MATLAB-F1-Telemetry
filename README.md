# 🏎️ F1 Telemetry Live Dashboard using MATLAB

A MATLAB-based animated telemetry dashboard inspired by Formula 1 race engineering systems.  
This project visualizes real-time vehicle parameters such as speed, engine RPM, steering input, and chassis vibration using dynamic plots and a custom dark-mode telemetry UI.

---

# 🚀 Features

- 📈 Real-time animated telemetry graphs
- 🏎️ Live digital speed display
- ⚡ Dynamic speed warning color system
- 🌌 Dark-mode telemetry dashboard styling
- 🎨 Custom RGB neon visualization
- 📊 Multi-plot telemetry layout
- 🔧 Signal simulation using trigonometric functions
- ⚙️ Chassis vibration visualization
- 🎥 Smooth MATLAB animation using `drawnow`
